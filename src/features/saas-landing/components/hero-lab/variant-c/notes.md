# Variant C Notes

## Layout concept

Variant C uses a wide split hero: dense but readable copy on the left, one dominant image on the right, and a framed visual shell that stretches further across desktop. The support content is intentionally reduced to a few high-signal panels so the page feels more open and the image carries more weight.

## Why this improves perceived quality

The broader image stage gives the hero more presence without adding visual noise. The larger desktop frame makes the layout feel intentional and premium, while the simplified support blocks keep the CTA and trust signals easy to scan. On mobile, the same structure collapses cleanly because the hero still has only one main visual and no micro-cards.

## Risks / tradeoffs

The stronger image treatment depends on the source photo having enough contrast and framing to survive a large crop. If the supplied hero image is too busy or too dark, the overlay and bottom caption may need tuning. The left/right balance is also more opinionated than the baseline hero, so it may feel too image-heavy on very large screens if the rest of the landing page stays text-led.
